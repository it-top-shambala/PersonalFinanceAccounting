Классы
1. CategoryExpense - категория расхода
    Свойства: Id - идентификатор категории, Name - название категории расхода
2. CategoryIncome - категории прихода
    Свойства: Id - идентификатор категории, Name - название категории прихода
3. Currency - валюта
    Свойства: Id - идентификатор валюты, Name - название валюты, Code - код валюты
4. Wallet - кошелек
    Свойства: Id - идентификатор кошелька, Name - название кошелька,
             Currency - объект класса валюты, Balance - количество средств в кошельке
    Методы:  public void Incoming(double sum) - пополнение кошелька, параметр sum - сумма пополнения
             public bool Take(double sum) - получение из кошелька, параметр sum - сумма получения
 5. LogOperation: Id - идентификатор лог-записи, DateTime - время проведения транзакции,
             NameCategory - название категории операции, Sum -  сумма транзакции
             (отрицательное значение - расход, положительное - приход)
