Классы
1. CategoryExpense - категории расходов
    Свойства: Id - идентификатор категории, Name - название категории расходов
2. CategoryIncome - категории приходов
    Свойства: Id - идентификатор категории, Name - название категории приходов
3. Currency - валюта
    Свойства: Id - идентификатор валюты, Name - название валюты, Code - код валюты
4. Wallet - кошелек
    Свойства: Id - идентификатор кошелька, Name - название кошелька,
              IdCurrancy - идентификатор валюты, Balance - количество средств в кошельке
    Методы:  public void Incoming(double sum) - пополнение кошелька, параметр sum - сумма пополнения
             public bool Take(double sum) - получение из кошелька, параметр sum - сумма получения
