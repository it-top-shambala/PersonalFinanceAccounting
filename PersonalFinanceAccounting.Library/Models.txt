Классы
1. CategoryExpense - категории расходов
    Свойства: Id - идентификатор категории, Name - название категории расходов
2. CategoryRefill - категории приходов
    Свойства: Id - идентификатор категории, Name - название категории приходов
3. Valuta - валюта
    Свойства: Id - идентификатор валюты, Name - название валюты, Code - код валюты
4. Wallet - кошелек
    Свойства: Id - идентификатор кошелька, Name - название кошелька,
              IdValuta - идентификатор валюты, Amount - количество средств в кошельке
    Методы:  public void Put(float sum) - пополнение кошелька, параметр sum - сумма пополнения
             public void Take(float sum) - получение из кошелька, параметр sum - сумма получения
