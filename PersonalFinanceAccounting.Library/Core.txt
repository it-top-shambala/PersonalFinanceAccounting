Интерфейс ICategoryCore:
    Методы:
        1- создание категории;
        2- изменение имени категории;
        3- удаление категории.
Классы:
*- CategoryExpenseCore: Класс для работы с расходными категориями. Наследуеться от интерфейса категорий. Наследуеться от интерфейса ICategoryCore.
      Переопереленные методы:
           1- создание категории - Статический метод создание новой категории расхода.
                                принимает имя категории и Возвращает объект класса Category.;
           2- изменение имени категории - Статический метод изменения имени категории расхода.
                                принимает старое и новое имя категории и Возвращает объект класса Category.;
           3- удаление категории - Статический метод удаления категории расхода.
                                принимает - имя категории.

*- CategoryIncomeCore: Класс для работы с доходными категориями. Наследуеться от интерфейса категорийнаследуеться от интерфейса ICategoryCore.
     Переопереленные методы:
           1- создание категории - Статический метод создание новой категории расхода.
                                принимает имя категории и Возвращает объект класса Category.;
           2- изменение имени категории - Статический метод изменения имени категории расхода.
                                принимает старое и новое имя категории и Возвращает объект класса Category.;
           3- удаление категории - Статический метод удаления категории расхода.
                                принимает - имя категории.

*- Transaction: Класс для обработки расходных и доходных транзакций

     Статические методы:
            1- метод проведения доходных транзакций. принимает  категорию транзакции, сумму транзакции, Кошелек.
            2- метод проведения расходных транзакций. принимает  категорию транзакции, сумму транзакции, Кошелек.

*- WalletCore Класс для работы с кошельком.
        Поля класса:
                1- Приватный список расходных категорий.
                2- Приватный список доходных категорий.
                3- Приватный кошелек.
                4- Приватный список валют.

        Конструктор по умолчанию WalletCore

        Методы:
                1- Метод возврата списка расходных категорий. Возвращает список объектов класса Category.
                2- Метод возврата списка доходных категорий. Возвращает список объектов класса Category.
                3- Метод возврата списка валют. Возвращает список объектов класса Currwncy.
                4- Метод создания кошелька. Принимает имя кошелька, валюту кошелька, баланс кошелька. Возвращает объект класса Wallet.
                5- Метод изменения имени кошелька. Принимает имя кошелька. Возвращает объект класса Wallet.
                6- Метод удаления кошелька. Принимает имя кошелька.
                7- Метод возврата баланса кошелька. Принимает имя кошелька. Возвращает (double) баланс кошелька.

*- Wallets Класс для возврата списка кошельков.
    
        Статический метод возврата списка кошельков. Возвращает список объектов класса Wallet.
